Creating ASCII Character Polygons with SQL Stored Procedure

For this assignment, I designed a stored procedure named sp_getLetter in SQL to dynamically generate single ASCII characters as polygons. These polygons serve a variety of purposes, including fixed-size labels, ocean labels, compass roses, and stylized monsters, catering to the needs of A'Tuin Co. and its clientele. Adhering to best practices, the procedure utilizes the geography object for programmatically creating planar polygons that exhibit appropriate distortion, particularly near the poles. By implementing vector math operations and ensuring planar geometry, the stored procedure offers a versatile solution for aesthetically pleasing text rendering in various contexts without relying on transactions, TRY CATCH blocks, or table references.
