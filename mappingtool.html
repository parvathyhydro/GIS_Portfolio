GeoMapper: Thematic Mapping Tool

For this project, I created a proportional symbol/cartogram map of all cities in Greenland using hexagons, focusing on thematic mapping to convey geographic feature properties effectively. I developed a WPF application capable of manipulating datasets (CSV or KML) to programmatically generate thematic maps. In addition to plotting hexagonal symbols for Greenland's cities, I added two features: a tool for generating random placemarks and another for calculating the sinuosity of river polylines from CSV data. These enhancements expanded the application's functionality, enabling users to create arbitrary geographic points and analyze river meandering. The project emphasized adherence to various metrics, user controls, error handling, and river analysis criteria, ensuring a comprehensive and functional solution.
