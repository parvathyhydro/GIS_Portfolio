Adapting SOPARC Field Tool with Smart Forms

For this project, I customized the System for Observing Play and Recreation in Communities (SOPARC) field data collection tool into a smart form compatible with ArcGIS Field Maps. Utilizing ArcGIS Pro, I created a point feature class to record park observations, meticulously aligning attribute fields with the SOPARC survey parameters. Implementing attribute domains and smart form functionalities, such as logical grouping, visibility logic, and different input types, ensured efficient data entry and streamlined user experience. Through rigorous testing and refinement, including the creation of fictional point features, I validated the form's functionality for field data collection, adhering to the SOPARC protocols outlined by Active Living Research.
