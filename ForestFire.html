Forest Fire Change Analysis from Satellite Imagery

In this assignment, utilizing Landsat 8 satellite imagery, I conducted change detection analysis focusing on the area around Lillooet and Lytton, British Columbia, impacted by the 2021 forest fire. Firstly, Landsat 8 data was collected for July 29, 2020, and September 2, 2021. Then, I created layouts showcasing true color and false color infrared composites for both time periods. Additionally, I examined the Normalized Difference Vegetation Index (NDVI) and Normalized Burn Ratio (NBR) index for the area, producing analytical maps illustrating changes post-fire. Each layout included explanatory text boxes elucidating the composite images and change detection analysis results, enhancing understanding and interpretation.
